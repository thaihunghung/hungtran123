Tích hợp thanh toán quét mã QR Code với MoMo, ViettelPay, VNPay, Vietcombank, Vietinbank, Techcombank, MB, ACB, VPBank, TPBank.. cho Woocommerce

== Description ==

- Không cần giấy phép kinh doanh.
- **Không yêu cầu nhập user/pass hay mã OTP, an toàn 100% !**
- Dùng được cho cả Tài khoản thụ hưởng là Số tài khoản doanh nghiệp.
- Xác nhận thanh toán & tự động kích hoạt đơn hàng từ 1~5 giây
- Xử lý đa luồng, không giới hạn số lượng giao dịch & kết nối website.
- Hỗ trợ QR code API tiêu chuẩn của Napas
- Hỗ trợ ví MOMO, ViettelPay, VNPay & các ngân hàng Vietcombank, Vietinbank, Techcombank, MB, ACB, VPBank, BIDV, TPBank, HDBank, Agribank, VIB, Sacombank, OCB, MSB, SHB, SCB, ABBank, BAC A Bank, Eximbank, Nam A Bank, NCB, SeABank, LienVietPostBank, Viet Capital Bank, Timo, Cake, TNEX..


**Yêu cầu:**

== Installation ==
Truy cập menu Plugins -> Add New và nhấn vào nút Install hoặc tải plugin về và giải nén vào thư mục wp-content/plugins rồi kích hoạt plugin.

Sau khi kích hoạt plugin, mở trang cấu hình WooCommerce > QHTestPay
